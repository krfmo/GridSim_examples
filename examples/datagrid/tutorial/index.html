OVERVIEW

What is gridsim.datagrid?

gridsim.datagrid is a Java package, an extension of the functionality of the Gridsim simulator. It enables the users to design and test different architectural approches  and different mechanisms which improve the availbility of the resources in the Data Grid.

How to use gridsim.datagrid?

  1. Program your own usage scenario in Java
  2. Use a set of  configuration files to define the topology and other parameters.
  3. Design your own architecture by extending the current classes .
